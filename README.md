# Stock Portfolio Tracker

## Project Description
A simple Python project that calculates the total investment value of a stock portfolio using predefined stock prices.

## Features
- Uses a dictionary to store stock prices
- Accepts user input for stock names and quantities
- Calculates investment value
- Displays total portfolio value
- Saves results to a text file

## Technologies Used
- Python

## Stock Prices Used

| Stock | Price |
